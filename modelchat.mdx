# ModelChat

ModelChat allows users to leverage our platform’s diverse range of models and generators. With ModelChat, users can create multimodal prompts with ease, combining text, images, and other media to generate rich and engaging content.

## Features

- **Diverse Models**: Access a wide variety of models for different tasks. These models are optimized for various applications and use cases.

- **Multimodal Prompts**: Combine text, images, and other media in your prompts. This allows for rich, context-aware interactions with the models.

- **Ease of Use**: Simple and intuitive interface for creating prompts. The interface is designed to be accessible for both beginners and advanced users.

## Getting Started

To get started with ModelChat, follow these steps within the Discord:

### Text Generation Models

Use the `#⁠model-chat` channel and follow the guide within the channel to use `/model` to access text-based & understanding model's private threads.

### Media Generation Models

Use the `⁠model-chat` channel's private thread for the following commands:
- `/text-to-video`
- `/text-to-image`
- `/image-to-video`

### Realtime Audio/Video Transcription

Go to the `⁠audio-chat` channel and use these commands:
- `/createvoice`
- `/talk`
- `/leave`

### Sound Effects & Music

Use `/text_to_sfx` to generate sounds and music based on the provided description and duration.

### Reference Commands

For all commands, refer to these help commands under each bot application:

- ModelChat: `/model_help`
- MediaGen: `/media_help`
- AudioGen: `/audio_help`

### Need Support?

DM/Ping `@StreamPod AI` or ask questions in `#⁠general-support`

## Example Usage

Here is an example of creating a multimodal prompt with ModelChat:

1. Use the `/model` command to select a text generation model
2. Upload an image using the chat interface
3. Prompt the model with the text and image to generate a response
